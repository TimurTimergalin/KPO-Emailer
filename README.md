# KPO-Emailer
Микросервис для отправки электронных сообщений

## API

### send

Запрос: `POST /send`

Тело запроса - json в текстовом виде, имеющий следующую структуру:

```json
{
    "receiver": <получатель>,
    "topic": <тема>,
    "text": <текст письма>,
    "filename": <имя файла>
}
```



Ответ:

```
{"status": "ok"}
```

если успешно, иначе возвращает ошибку: 404, если файл не найден
